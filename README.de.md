lock_inactive_users
===================

Beschreibung
------------

Dies ist ein Modul für UliCMS, welches User-Accounts, die sich länger als X Tage
nicht in das UliCMS Backend eingeloggt haben, automatisch sperrt.

Voraussetzungen
---------------

-   UliCMS 2018.4 oder neuer

-   better_cron 1.1 oder neuer
